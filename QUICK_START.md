## 🚀 部署步骤 / Deployment Steps

### 第1步：准备服务器 (Prepare Server)
```bash
# 1. 登录服务器
ssh user@server.com

# 2. 创建网站目录
mkdir -p /var/www/conmotwu.com
cd /var/www/conmotwu.com

# 3. 上传文件 (从本地)
scp -r conmotwu.com/* user@server:/var/www/conmotwu.com/
```

### 第2步：配置权限 (Set Permissions)
```bash
# 设置目录权限
chmod -R 755 /var/www/conmotwu.com/

# 设置文件权限
chmod 644 /var/www/conmotwu.com/*.html
chmod 644 /var/www/conmotwu.com/css/*
chmod 644 /var/www/conmotwu.com/js/*
chmod 644 /var/www/conmotwu.com/images/*
chmod 644 /var/www/conmotwu.com/.htaccess
```

### 第3步：启用HTTPS (Enable HTTPS)
```bash
# 安装Certbot
sudo apt-get install certbot python3-certbot-apache

# 获取Let's Encrypt证书
sudo certbot certonly --apache -d conmotwu.com -d www.conmotwu.com

# 启用模块
sudo a2enmod ssl
sudo a2enmod rewrite
sudo systemctl restart apache2
```

### 第4步：配置Apache虚拟主机 (Configure VirtualHost)
```apache
<VirtualHost *:443>
    ServerName www.conmotwu.com
    ServerAlias conmotwu.com
    DocumentRoot /var/www/conmotwu.com
    
    SSLEngine on
    SSLCertificateFile /etc/letsencrypt/live/conmotwu.com/fullchain.pem
    SSLCertificateKeyFile /etc/letsencrypt/live/conmotwu.com/privkey.pem
    
    <Directory /var/www/conmotwu.com>
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>

# HTTP to HTTPS redirect
<VirtualHost *:80>
    ServerName www.conmotwu.com
    ServerAlias conmotwu.com
    Redirect 301 / https://www.conmotwu.com/
</VirtualHost>
```

### 第5步：配置DNS (Configure DNS)
```
A记录: conmotwu.com → 你的服务器IP
CNAME: www → conmotwu.com
```

## 📞 技术支持 / Technical Support

### 常见问题解决 (Troubleshooting)

**Q: 网站显示404错误**
A: 检查.htaccess文件是否正确上传和启用

**Q: HTTPS显示不安全警告**
A: 重新生成或更新SSL证书

**Q: 表单不发送邮件**
A: 检查PHP邮件功能是否启用，或配置第三方邮件服务

**Q: 移动设备上导航破损**
A: 检查CSS/style.css中的媒体查询是否正确
